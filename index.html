Conocimientos

Montaje y configuración de servidores Linux con stack LAMP, configuración de protocolo SSH, administración remota. Gestión de servidor web, montaje de sitios, configuracion de scripts, certificados de seguridad. Registros de dominios y pericia en contratación de hostings. Administración de redes, protocolos de seguridad, configuración de puertos. Trabajo en desarrollo en entornos de programación con conocimientos en lenguajes HTML5, CCS3, JavaScript, Python. Uso de GIT y GitHub. Manejo de bases de datos MySQL y Firebase. Colaboracion en implementacion y administración de sistemas ERP y CRM tales como Odoo, Bitrix24 y Monday. Proyectos relacionados a IoT y robotica, conocimiento sobre distintos devs kits (Arduino, Espressif, Raspberry),componentes electronicos, esquemas de desarrollo y programación.

Proyectos

Actualmente me encuentro trabajando en algunos proyectos personales, a modo de aprendizaje y cuestiones relacionados.

Desarrollo web personal

Se trata del sitio que usted se encuentra visualizando. Actualmente trabajo en integrar distintos scripts y mejoras que permitan darle una funcionalidad más profesional al mismo ademas de funcionar tambien como una carta de presentación personal.

Servidor web personal

En donde se aloja el sitio web que usted se encuentra visualizando. Monté mi propio web hosting haciendo uso de una placa Raspberry pi 2 modelo B con sistema basado en Debian. Utilizando el stack LAMP, es decir ademas de Linux (Debian), servidor Apache, MySQL y PHP. Tambien he instalado y configurado certificados SSL y protocolo de seguridad SSH (par de llaves).

Proyecto de robotica

Buscando realizar una integracion de las funcionalidades web de la misma para tomar datos de medición y almacenarlos en una base de datos a su vez de desarrollar una aplicacion movil sencilla a modo de compañera.
